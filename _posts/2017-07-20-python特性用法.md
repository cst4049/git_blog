zip打包解包列表和倍数
```python
a = [1,2,3]
b = ['a','b','c']
z = zip(a,b) #[(1,'a'),(2,'b'),(3,'c')]
zip(*z)      #[(1,2,3),("a",'b','c')]

```
zip合并相邻的列表项
```python
a = [,1,2,3,4,5,6]
zip(*([iter(a)] * 2)) #[(1,2),(3,4),(5,6)]
```
