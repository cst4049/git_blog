##深浅拷贝对比
```python
  import copy
  will = ["Will", 28, ["Python", "C#", "JavaScript"]]
  wilber = copy.copy(will)
  wilber1 = copy.deepcopy(will)
  will[2].append("hello world")

  print (will)
  print(wilber)
  print(wilber1)
```
##输出对比
```javascript
   ['Will', 28, ['Python', 'C#', 'JavaScript', 'hello world']]
   ['Will', 28, ['Python', 'C#', 'JavaScript', 'hello world']]
   ['Will', 28, ['Python', 'C#', 'JavaScript']]
```
